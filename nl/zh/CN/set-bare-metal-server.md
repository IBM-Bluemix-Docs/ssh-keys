# 配置裸机服务器

1. 登录到 {{site.data.keyword.slportal}}
* 单击“设备”>“设备列表”
* 在“设备列表”中找到您的系统
* 选择在“快照”视图中或“设备详细信息”屏幕上管理服务器
* 记录服务器的 IP 地址和凭证
* 更新操作系统和其他软件的凭证
* 访问专用网络上的服务器
## 后续操作
设置服务器之后，请完成任务以设置监视、确保服务器安全以及安排备份。
