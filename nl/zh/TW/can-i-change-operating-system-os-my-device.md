# 變更裝置上的作業系統 (OS)

您可以變更在執行時所安裝的 OS 及軟體。在裝置上選取「OS 重新載入」之後，即會提供鏈結來編輯系統上的軟體。您可以使用這項編輯來更新 OS、「控制面板」、「防毒」套件及資料庫軟體。

如需相關資訊，請參閱 [OS 重新載入](../infrastructure/software/vsi_reload_os.html#reloading-an-os)。
