# 配置裸機伺服器

1. 登入 {{site.data.keyword.slportal}}
* 按一下「裝置」>「裝置清單」
* 在「裝置清單」中，尋找系統
* 選擇在「Snapshot 視圖」中或「裝置詳細資料」畫面上管理伺服器
* 記錄伺服器的 IP 位址及認證
* 更新作業系統及其他軟體的認證。
* 在專用網路上存取伺服器
## 接下來會發生什麼情況
設定伺服器之後，請完成作業來設定監視、保護伺服器安全，以及排定備份。
